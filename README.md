# NYC Taxi Trip Duration

Machine Learning I (DATS 6202) - George Washington University

Kaggle competition: [New York City Taxi Trip Duration](https://www.kaggle.com/c/nyc-taxi-trip-duration)

## Project Structure

```
.
├── notebooks/
│   ├── homework_1.ipynb      # Homework 1 - Training Shallow Models
│   └── case_study.ipynb      # Reference case study (House Prices)
├── data/                     # Dataset (not tracked by git)
│   ├── train.csv
│   ├── test.csv
│   └── sample_submission.csv
├── requirements.txt
└── README.md
```

## Setup

1. Download the dataset from [Kaggle](https://www.kaggle.com/c/nyc-taxi-trip-duration/data) and place it in the `data/` directory.

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
